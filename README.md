# JSON-Slicer-for-FAQ
Smart FAQ to JSON converter. Supports English and Chinese text processing. | 智能 FAQ 转 JSON 工具，支持中英文文本处理。

### 🛠 如何安装
1. 下载此仓库并解压。
2. 打开 Chrome 浏览器，进入 `chrome://extensions/`。
3. 开启右上角的 **“开发者模式”**。
4. 点击 **“加载已解压的扩展程序”**，选择解压后的文件夹。

### 📖 如何使用
1. 在 Google Docs 或网页中复制 FAQ 文本。
2. 点击插件图标。
3. 在上方框内粘贴文本，下方将自动生成 JSON。
4. 点击“复制结果”即可使用。
