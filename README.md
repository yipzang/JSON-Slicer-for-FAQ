# JSON-Slicer-for-FAQ
Smart FAQ to JSON converter. Supports English and Chinese text processing. | 智能 FAQ 转 JSON 工具，支持中英文文本处理。

📝 Description / 插件简介

EN:
A lightweight Chrome extension designed for productivity. It accurately splits FAQ (Frequently Asked Questions) text into structured JSON format by identifying question marks. Perfectly compatible with Google Docs and web content. No background processes—privacy-focused and efficient.

ZH: 
一款专为提高生产力设计的轻量级 Chrome 插件。它通过精准识别问号，将杂乱的问答文本（FAQ）自动切割并转换为标准 JSON 格式。完美兼容 Google Docs 和各类网页内容。无需后台常驻进程，保护隐私且高效运行。

---

✨ Features / 功能亮点
Bilingual Support: Optimized for both Chinese and English sentence structures. (支持中英文句式优化)
Zero Permission Issues:Uses a paste-and-convert method to bypass complex web restrictions. (采用粘贴转换模式，绕过网页权限限制)
Smart Detection:Automatically cleans up whitespace and handles multi-line answers. (自动清理空格，支持多行答案识别)
